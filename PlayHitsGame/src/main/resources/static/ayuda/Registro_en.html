Para realizar el registro, debes cumplimentar el campo Usuario, con un mail valido, al que te llegara un correo con un link que deberás usar, para validar el registro, como medida de seguridad, para evitar altas indiscriminadas.

Cumplimenta el campo de contraseña y su confirmación, intenta usar mayúsculas, minúsculas, números y caracteres especiales, para hacerla lo mas segura posible.

El campo Grupo, se puede cumplimentar con cualquier tipo de caracteres. Es usado para poder hacer partidas con otros jugadores, es decir todos los jugadores que indiquen el mismo nombre de grupo, podrán hacer una partida de Grupo juntos. En el registro lo puedes dejar en blanco, posteriormente, lo podrás cumplimentar en el momento que lo desees.

Finalmente puedes clickar el botón enviar, con el que recibirás un correo en el correo indicado en Usuario, comprueba la carpeta spam o correo no deseado, en el que deberás clickar el enlace para confirmar el registro.

Una vez confirmado el registro, puedes cliclar el link Acceder, o volver a la pagina de Inicio, para volver al login y usar el correo indicado como usuario y la contraseña para acceder a la aplicación.
